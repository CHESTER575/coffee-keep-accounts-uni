# 咖啡记账

## 项目介绍

一款用于记录每天开销的小程序

## 技术栈

uni-app + vue3 + typescript + pinia

## 项目克隆

    git clone https://github.com/CHESTER575/coffee-keep-accounts-uni

## 依赖安装

    npm install

## 运行

### 运行到H5

    npm run dev:h5

### 运行到小程序

    npm run dev:mp-weixin

## 注意事项

此项目仅用来学习uni-app、vue3、typescript、css预处理器
